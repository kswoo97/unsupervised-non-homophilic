For the below datasets, PyG automatically loads dataset:
{Cora, Citeseer, Pubmed, Photo, Computers, Chameleon, Squirrel, Actor, Cornell, Wisconsin, Texas}

However, for the datasets {Arxiv, Penn94, Flickr}, one should download the pre-processed datasets from the following link:
https://drive.google.com/drive/folders/1YugoLW4wV8--NXa92MiKxVK7UcrzWUnA?usp=sharing
Files should be
- {data_name}.pt
- {data_name}_pre.pt
